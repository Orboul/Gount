# Tracker Setup:


This documents how to set up the tracker

first you need the Binary, you download that to the server where you want the tracker to live for example you vps.

next you start and configure the program using the setup.sh script

then you need to use your dns and point the program servecie to where you need it for example your domain name like "tracker.example.com", once you have that setup then you just need to place the tracker code or pixel tracker in your webpage, making sure to use the correct ip or doamin name.
